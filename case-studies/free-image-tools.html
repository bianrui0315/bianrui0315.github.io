Free Image Tools

Rui built a free, open-source image utility suite where core compression, conversion, resizing, and PDF workflows run in the browser, reducing privacy friction for everyday creator and developer tasks.

The Product Problem

Many online image utilities require users to upload files for simple transformations. Rui positioned Free Image Tools around a clearer promise: common image work should be fast, free, inspectable, and private by default.

Rui's Role

Rui built the product surface, browser processing flows, Cloudflare deployment, SEO-oriented tool pages, GitHub repository, and collaboration-friendly open-source positioning.

What Rui Built

  • Browser-first processing: compression, resizing, format conversion, and batch workflows run locally with Web APIs.
  • Privacy-centered UX: no upload required for core tools, no server-side file storage, and metadata-removal workflows for safer sharing.
  • Useful tool coverage: WebP/AVIF conversion, target-size compression, social presets, PDF-to-image, image-to-PDF, Base64 output, borders, circular crops, and palette extraction.
  • Optional AI layer: AI alt-text generation supports accessibility and SEO without making the core product depend on account creation or paid tiers.

Open Collaboration

The website is free to use and the repository is public. Rui invites developers and users to report issues, suggest tools, open pull requests, and improve the project together.
